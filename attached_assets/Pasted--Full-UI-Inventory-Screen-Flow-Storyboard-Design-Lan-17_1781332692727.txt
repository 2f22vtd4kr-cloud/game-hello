**Design Language**: Apple-style Liquid Glass + Holographic Futuristic (translucent frosted glass cards, neon cyan/purple glows, deep space dark background, 3D depth, reflections, subtle animations)

### 1. UI INVENTORY (Comprehensive Component Library)



#### Core Visual Elements
- Background: Deep navy/black gradient with subtle star/constellation particles + reflective floor
- Glass Cards: Frosted translucent panels with blur, thin neon borders (cyan/purple), inner glow
- Accents: Electric cyan, magenta-purple, white highlights
- Icons: Minimal line icons (fuel pump, map pin, trophy, radar) with glow
- Typography: SF Pro / Inter — Bold headings, clean body



#### Global Components
- Header Bar: Holographic top bar with app title "ТОПЛИВНЫЙ УЗЕЛ", back button, Telegram Mini App close
- Bottom Navigation: 4-tab glass bar — Карта | Резервы | Игры | Профиль
- Floating Action Button (FAB): Glowing fuel pump icon for quick "Report Fuel"
- Neon Buttons: Gradient fill (cyan→purple), hover/press scale + glow
- Progress Bars: Liquid-filled horizontal bars with percentage
- Status Dots: Green (available), Yellow (talons), Red (no fuel), Gray (unknown)



#### Reusable Components
- Station Card: Glass panel with station name, fuel grid (progress + ✓/✗), queue indicator, "Subscribe" bell button
- Fuel Indicator: Circular or horizontal gauge with fuel type (AI-95 etc.)
- Map Marker: Glowing holographic pins with color-coded rings
- Notification Toast: Sliding glass card from top with fuel alert
- Level Badge: Shield + XP counter with particle effect on level up
- Mini-Game Panel: Fuel pump tapping area with timer and score



#### Data Visuals
- Regional Reserves Dashboard (bars for AI-95, AI-92, DT, etc.)
- Activity Heatmap on Russia/Crimea map
- Leaderboard cards with ranks and avatars

### 3. STORYBOARD (Sequential User Journey)



#### Screen 1: Splash / Onboarding
- Full-screen animated liquid glass sphere with glowing fuel pump icon
- Text: "ТОПЛИВНЫЙ УЗЕЛ" + "Матрица Снабжения"
- Subtle rotating holographic tanks + robotic arms in background
- "Войти через Telegram" button (glow pulse)



#### Screen 2: Home — Карта (Main View)
- Full-screen Leaflet map of Russia/Crimea with glowing markers
- Top holographic panel: Current reserves summary (AI-95 55%, etc.)
- Bottom nav active on "Карта"
- Floating "Report" FAB
- Floating "Radar" button (pin icon)



#### Screen 3: Station Detail (Modal)
- Large glass card sliding up
- Station name + network logo
- Fuel grid with liquid progress bars + status buttons (✓ / ✗)
- Queue counter + "Очередь" button
- "Подписаться на обновления" glowing bell
- QR code section (for official stations)



#### Screen 4: Резервы (Reserves Dashboard)
- Top: Regional liquid glass overview with progress bars
- Middle: Interactive map of active zones (green/yellow/red)
- Bottom: City list (Симферополь, Севастополь, etc.) with status



#### Screen 5: Игры & Награды
- "Заправка на скорость" mini-game preview
- Daily leaderboard (glass cards with ranks)
- Reward circles: 65% fuel found, 25% reports, etc.
- Trophy case with unlocked badges



#### Screen 6: Профиль
- Large user level shield ("Новичок" → "Легенда Тавриды")
- XP progress ring
- Stats: Reports made, fuel helped find, friends invited
- History feed of contributions



#### Additional Flow Screens
- Notification Screen: List of "Fuel appeared at ТЭС-12!"
- Radar Active: Map with pulsing radius circle + live alerts
- Error / Empty States: Elegant glass "No data" with retry
- Level Up Animation: Particle explosion + glowing shield upgrade

Design System Summary (Liquid Glass Style)



- All cards have backdrop-filter: blur(20px), semi-transparent white/black base, neon border glow
- Heavy use of depth (shadows + inner highlights)
- Micro-animations: fuel liquid fill, marker pulse, button press haptic + scale
- Consistent icon set: Fuel pump, map, bell, trophy, robot arm accents
- Color palette: #00F0FF (cyan), #C026D3 (purple), dark navy background
